# Spotify Web Player Clone

A responsive **Spotify-inspired Web Player UI** built using **HTML & CSS**.
This project focuses on layout design, flexbox, UI structure, and styling — perfect for frontend practice.

---

## Features

* Sidebar navigation (Home, Search, Library)
* Playlist & podcast sections
* Music cards (Recently Played, Trending, Featured)
* Sticky navigation bar
* Music player UI (controls + progress bar)
* Responsive design (basic media queries)
* Clean dark theme inspired by Spotify

---

## Tech Stack

* **HTML5**
* **CSS3**
* **Font Awesome Icons**
* **Google Fonts (Montserrat)**

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/spotify-clone.git
```

2. Open the project folder:

```bash
cd spotify-clone
```

3. Run the project:

* Open `index.html` in your browser.
